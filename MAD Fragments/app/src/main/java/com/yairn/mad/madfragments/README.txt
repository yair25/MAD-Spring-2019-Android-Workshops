These are some instructions on how to make this application.
There are also some hints/suggestions on what to say to help the understanding.

Steps:
1. Start with app > res > layout > activity_main.xml
2. With FrameLayout, add the Layouts and views
3. Next go to app > java > com.________ > MainActivity
4. Code and link to UI

5. Next, make ListFragment
6. Add ListView to fragment_list.xml
7. Make item_view.xml and add TextView
8. Code ListFragment

9. Next make PicsFragment
10. Add ImageView to fragment_pics.xml
11. Add Images to drawable from
    https://drive.google.com/open?id=1z9lW_R9WVciaVGa4Ta-6qNlrcNC5amcF
12. Code PicsFragment

13. Next make FullPicFragment
14. Add ImageView to fragment_full_pic.xml
15. Code FullPicFragment

16. Finish up MainActivity
